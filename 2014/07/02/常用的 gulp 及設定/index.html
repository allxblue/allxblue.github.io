    • 常用的 gulp plugin 整理

    其實之前有使用Fire.app的，不過在windows底下SASS存檔編譯的速度實在太慢了，配合LiveReload來說用起來很不順手，剛好在前端聚會聽大大提起後，立刻開始研究。

    這篇將常用的列出來，下一篇寫配置

    常用的gulp plugin

    擺在上面供查詢，若有推薦的也可以留言分享

    • gulp
    • gulp-concat - 最後輸出用
    • gulp-uglify - 壓縮js
    • gulp-imagemin - 壓縮圖片
    • gulp-autoprefixer - css自動修正
    • gulp-minify-css - 最小化css
    • gulp-sass - 處理SASS
    • gulp-compass - 需要裝Ruby，也是用來處理SASS
    • gulp-watch - 偵聽檔案、資料夾變化
    • gulp-livereload - 就livereload
    • gulp-gzip - 檔案自動gzip壓縮，不過通常都由server統一處理了
    • gulp-zip - 壓縮用，配合exec可以做很多事
    • gulp-clean - 刪除檔案
    • gulp-exec - 透過他執行指令，用這個混搭幾乎就是一鍵佈署
    • gulp-if - 三元判斷用
    • gulp-order - 排序stream用，不過我個人測試都失敗 囧
    • gulp-notify - 跳出提醒用，windows要裝Growl for Windows
    • gulp-rename - 重新命名用
    • gulp-plumber - 報錯後仍繼續執行

    下面是使用 npm install gulp-xxx --save 存下來的package.json npm 配置
